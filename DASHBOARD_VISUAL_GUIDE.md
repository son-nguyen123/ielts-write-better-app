## Color Scheme

### Primary Elements (Blue)
- Overview card icons
- Target setting highlights
- "Focus on These" card background (gradient)
- Practice goal card
- Links and buttons

### Success Elements (Green)
- Best recent score icon
- Positive progress indicators
- Achieved targets

### Warning Elements (Orange/Yellow)
- Gap to target (when positive gap)
- Medium priority badges
- Pro tip sections

### Destructive Elements (Red)
- High priority badges
- Critical skill gaps
- Worsening trends

### Neutral Elements (Gray)
- Card backgrounds
- Borders
- Muted text
- Secondary information

## Visual Hierarchy (Top to Bottom)

1. **Level 1 - Header**: User greeting + target
2. **Level 2 - Metrics**: 4 overview cards (most important numbers)
3. **Level 3 - Goal Setting**: Target + Skill gaps (what user wants vs. has)
4. **Level 4 - Action Items**: "Focus on These" (MOST PROMINENT - gradient background)
5. **Level 5 - Study Plan**: How to practice effectively
6. **Level 6 - Progress**: Charts and recent activity
7. **Level 7 - Quick Actions**: CTAs to start practicing

## Eye-Catching Elements

### Large Text
- "FOCUS ON THESE TO IMPROVE FAST!" (2xl font)
- Score numbers (3xl font in overview cards)
- Gap indicators (+2.2, etc.)

### Color Contrast
- Gradient card with border (primary color)
- High-contrast numbered badges (â‘  â‘¡ â‘¢ â‘£)
- Colored priority badges (ğŸ”´ high, ğŸŸ¡ medium, ğŸŸ¢ low)

### Iconography
- Emojis for quick visual recognition
- Lucide icons for actions
- Badges for categorization

### Spacing & Borders
- Generous padding in focus cards
- 2px borders on important cards
- Rounded corners (rounded-2xl)
- Hover effects for interactivity

## Responsive Behavior

### Desktop (lg: 1024px+)
- 4 overview cards in a row
- Target setting (1/3) + Skill priority (2/3)
- 2x2 grid for focus issues
- 2x1 grid for study plan actions
- 2 columns for progress chart + recent activity

### Tablet (md: 768px+)
- 2 overview cards per row
- Target setting full width, then skill priority
- 2x2 grid maintained for focus issues
- 2x1 grid for study plan
- Stacked progress + activity

### Mobile (<768px)
- 1 overview card per row (stacked)
- All sections full width
- Focus issues stacked vertically
- Study plan cards stacked
- All content in single column

## Loading States

All sections show:
- Spinner icon (Loader2)
- "Loading..." or descriptive text
- Skeleton UI for placeholders
- Graceful fallback if data unavailable

## Empty States

- No essays: Shows basic stats + prompt to start
- No target: Prompts user to set target
- No recurring issues: Shows encouraging message
- No activity: Suggests creating first task
